### Задание 1. Кеширование 

Приведите примеры проблем, которые может решить кеширование. 

*Приведите ответ в свободной форме.*

# Ответ: 

Пример №1. Высокая нагрузка на базы данных

 Есть проблема: При каждом запросе пользователя система обращается к базе данных (БД). Если сайт популярный, БД начинает задыхаться от количества запросов, что приводит к медленной работе или падению.

Решение: Результаты частых запросов (например, список товаров или профиль пользователя) сохраняются в быстром кеше (например, Redis или Memcached). При повторном запросе данные отдаются из кеша, минуя БД, что снижает нагрузку на неё в разы. 

Пример №2  Медленная загрузка страниц или данных

 Есть проблема: Генерация главной страницы, например, интернет-магазина требует сложных вычислений, сборки данных из 10 разных источников или обработки изображений. Каждый раз генерировать её заново — долго.

 Решение: Готовая HTML-страница или результат сложного вычисления сохраняется в кеше. Следующий пользователь получает уже готовую страницу мгновенно.
